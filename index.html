World Wide Scan

PROJECT DESCRIPTION

With the development of technology, technological devices and internet have become indispensable for our lives. Although this provides many advantages, it poses a threat to personal data security. The aim of our project is to detect these threats quickly and easily. In this way, it is aimed to detect weaknesses in the system and to eliminate them in the relevant institution as soon as possible.

Services

Subdomain Discovery

System Version Detection & Vulnerability Scan

Assessment of Detected Vulnerabilities

PROBLEM

Some of the web servers used by educational institutions on their systems are not updated in a controlled manner. Therefore, security gaps occur in web systems. These vulnerabilities may cause system degradation and data disclosure.

SOLUTION

First of all, the subdomains of the relevant institution are determined. Afterwards, detected subdomains will be programmed in parallel and vulnerability scanning will be done with advanced search artifacts. By taking the latest, a statistics will be published by comparing web service versions in more than 50 countries in 10 different countries.
